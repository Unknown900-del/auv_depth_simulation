# AUV Underwater Simulation

This project simulates an Autonomous Underwater Vehicle (AUV) navigating underwater using a physics-based model and control systems.

## Requirements

To run the simulation:

pip install numpy matplotlib

## Features

- Physics-based AUV motion
- Buoyancy, gravity, drag and thrust forces
- Depth stabilization using PID control
- Sonar-based obstacle detection
- Fish ecosystem environment
- Seabed and wave visualization
- Trajectory path visualization

## Concepts Used

- Hydrodynamics
- Control Systems
- PID Controller
- Sonar sensing
- Obstacle detection

## Technologies Used

- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## Simulation Components

- AUV vehicle model
- Rotating sonar beam
- Fish acting as dynamic obstacles
- Water surface waves
- Seabed terrain
