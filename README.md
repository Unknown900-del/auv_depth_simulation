# AUV Underwater Simulation

This project simulates an Autonomous Underwater Vehicle (AUV) using Python.

Features:
- AUV physics model (gravity, buoyancy, drag)
- PID depth control
- Sonar obstacle detection
- Fish ecosystem
- Water surface waves
- Seabed environment
- Trajectory visualization

Technologies:
- Python
- Matplotlib
- Numpy
- Jupyter Notebook
